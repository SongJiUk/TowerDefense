# TowerDefense — 게임 기획서

> 작성일: 2026-04-15
> 장르: 3D 로그라이크 타워디펜스 / 플랫폼: 모바일 (Android / iOS)

---

## 1. 게임 개요

| 항목            | 내용                                                   |
| --------------- | ------------------------------------------------------ |
| 장르            | 3D 로그라이크 타워디펜스                               |
| 시점            | 2.5D (카메라 45도 고정)                                |
| 플랫폼          | 모바일 (Landscape Left 고정, 1920×1080)                |
| 목표 플레이타임 | 1런 20~30분                                            |
| 핵심 재미       | 타워 시너지 조합 + 웨이브마다 달라지는 로그라이크 선택 |

### 핵심 루프

```
웨이브 시작
  └→ 적이 A* 경로를 따라 코어로 이동
  └→ 타워 배치 / 업그레이드로 적 처치
  └→ 웨이브 클리어
        └→ 로그라이크 선택 (3택 1)
        └→ 다음 웨이브 준비
  └→ 코어 HP 0 → 런 종료 (패배)
  └→ 스테이지 10웨이브 클리어 → 다음 스테이지
```

---

## 2. 스테이지 구성

### 전체 구조

```
4개 스테이지 × 10웨이브 = 총 40웨이브
```

### 스테이지별 테마 & 난이도

| #   | 테마    | 기준 HP 배율 | 특징                          |
| --- | ------- | ------------ | ----------------------------- |
| 1   | 🌲 숲   | ×1.0         | 튜토리얼 느낌, 직선 경로      |
| 2   | 🏜 사막 | ×1.6         | 모래嵐 구간 (감속 타워 효율↑) |
| 3   | ❄️ 겨울 | ×2.5         | 빙판 구간 (적 이동속도 보정)  |
| 4   | 🔥 던전 | ×3.8         | 분기 경로, 가장 복잡한 맵     |

### 스테이지 클리어 보상

- 코어 HP **+2 회복**
- 다음 스테이지 시작 전 **무료 타워 업그레이드 1회** 선택

## 3. 웨이브 설계

### 웨이브 구간 (스테이지당)

| 구간 | 웨이브 | 역할                           |
| ---- | ------ | ------------------------------ |
| 입문 | 1~3    | 적 소수·느림. 타워 배치 익히기 |
| 중반 | 4~6    | 적 종류 다양화, 물량 증가      |
| 후반 | 7~9    | 고속 적 + 탱커 혼합            |
| 보스 | 10     | 보스 1마리 + 잡몹              |

### 웨이브별 스케일링 공식

```
HP        = 기준HP × (1 + 웨이브번호 × 0.15)
            → Wave 1: ×1.15 / Wave 10: ×2.50
이동속도   = 기준속도 (Wave 5, Wave 8에서 각 +0.1 추가)
스폰 수    = 3 + 웨이브번호 × 2
            → Wave 1: 5마리 / Wave 10: 23마리
```

### 웨이브 타이밍

| 항목                  | 값                             |
| --------------------- | ------------------------------ |
| 웨이브 시작 대기      | 5초 (선택 UI 완료 후)          |
| 스폰 간격             | 1.5초 (후반 웨이브 1.0초)      |
| 보스 웨이브 스폰 간격 | 3.0초 (보스 입장 연출 후 잡몹) |

## 4. 적 (Enemy) 설계

### 적 종류 5종

| 종류          | HP   | 이동속도 | 처치 보상 | 특성                     |
| ------------- | ---- | -------- | --------- | ------------------------ |
| 기본 (Basic)  | 80   | 2.5      | 3G        | 없음                     |
| 탱커 (Tank)   | 300  | 1.5      | 8G        | 방어력 20%               |
| 고속 (Runner) | 50   | 5.0      | 4G        | 감속 저항 50%            |
| 비행 (Flyer)  | 60   | 3.5      | 5G        | 지상 타워 일부 공격 불가 |
| 보스 (Boss)   | 1500 | 1.8      | 50G       | 처치 시 웨이브 종료      |

> **비행 적 참고:** Slow·Poison 타워는 공격 가능. Basic·Sniper는 공격 불가 (구현 단계에서 조정 가능)

### 스테이지별 등장 적

| 스테이지 | 주요 등장             |
| -------- | --------------------- |
| 1 숲     | Basic, Tank           |
| 2 사막   | Basic, Runner, Tank   |
| 3 겨울   | Runner, Tank, Flyer   |
| 4 던전   | 전 종류 + 보스 강화형 |

## 5. 골드 경제

### 수입

| 항목                 | 골드                |
| -------------------- | ------------------- |
| 시작 골드            | 150                 |
| 웨이브 클리어 보너스 | 30 + 웨이브번호 × 5 |
| 적 처치 보상         | 종류별 3~8G         |
| 보스 처치 보상       | 50G                 |

> Wave 1 클리어 기준 수입 예시:
> 적 5마리 × 3G(15G) + 클리어 보너스 35G = **50G**
> → 기본 타워(50G) 1개 설치 가능한 경제 밸런스

## 6. 타워 (Tower) 설계

### 타워 6종 기본 스탯

| 타워      | 공격력 | 공속  | 사거리 | 타겟 방식         | 특성                  |
| --------- | ------ | ----- | ------ | ----------------- | --------------------- |
| Basic     | 20     | 1.0/s | 4      | 단일 (최근접)     | 없음                  |
| Cannon    | 35     | 0.6/s | 3.5    | 범위 (반경 2m)    | 스플래시              |
| Slow      | 8      | 1.2/s | 3.5    | 단일              | 이동속도 -40%, 2초    |
| Sniper    | 80     | 0.3/s | 7      | 단일 (최원거리)   | 방어력 무시           |
| Poison    | 5/tick | 1.5/s | 3.5    | 단일              | DoT 4초 (중첩 가능)   |
| Lightning | 25     | 0.8/s | 4      | 체인 (최대 3마리) | 체인 피해 -30%씩 감소 |

### 업그레이드 배율 (Lv1 → Lv2 → Lv3)

| 항목   | Lv2  | Lv3  |
| ------ | ---- | ---- |
| 공격력 | ×1.4 | ×2.0 |
| 공속   | ×1.2 | ×1.5 |
| 사거리 | +0.5 | +1.0 |

## 7. 시너지 시스템

인접한 두 타워가 같은 타일 그룹에 있을 때 자동 발동.

| 시너지    | 조합               | 효과                                 |
| --------- | ------------------ | ------------------------------------ |
| 저격 집중 | Slow + Sniper      | 감속 상태 적에 Sniper 데미지 ×1.8    |
| 독 확산   | Slow + Poison      | 감속된 적 사망 시 주변 2칸에 독 전파 |
| 폭발독    | Cannon + Poison    | Cannon 범위 적중 시 독 스택 1 추가   |
| 과부하    | Lightning + Basic  | 체인 이후 인접 Basic 공속 +50% (3초) |
| 감전      | Sniper + Lightning | Sniper 피격 적에 Lightning 자동 체인 |

> 시너지 발동 시 타워에 아이콘 표시 → 플레이어가 조합을 인지할 수 있도록

## 8. 로그라이크 선택 시스템

레벨업시 **3개 선택지를 1개 선택** (항상 3개 카테고리 각 1개 보장)

### 카테고리 A — 전투 강화

| 선택지      | 효과                      |
| ----------- | ------------------------- |
| 화력 집중   | 모든 타워 공격력 +12%     |
| 속사        | 특정 타워 종류 공속 +25%  |
| 사거리 연장 | 모든 타워 사거리 +1       |
| 치명타      | 타워 15% 확률로 데미지 ×2 |

### 카테고리 B — 경제

| 선택지        | 효과                                 |
| ------------- | ------------------------------------ |
| 즉시 지원     | 골드 +80 즉시 획득                   |
| 풍요          | 이번 스테이지 처치 보상 +50%         |
| 할인          | 타워 설치비 -20% (스테이지 종료까지) |
| 웨이브 보너스 | 다음 웨이브 클리어 보너스 ×2         |

### 카테고리 C — 특수

| 선택지      | 효과                           |
| ----------- | ------------------------------ |
| 무료 배치   | 타워 1종 즉시 무료 설치        |
| 선제 공격   | 다음 웨이브 적 HP -20% 시작    |
| 코어 강화   | 코어 HP +3 회복                |
| 시너지 증폭 | 발동 중인 시너지 효과 +30%     |
| 타워 이전   | 배치된 타워 1개 위치 이동 가능 |

### 카테고리 D - 스킬

| 선택지           | 효과                                                                                      |
| ---------------- | ----------------------------------------------------------------------------------------- |
| 스킬 선택        | 스킬 1종 선택 가능(3가지 모두 가득 찼을때 교체 가능 - 패널티있음(스킬레벨 이어가지 않음)) |
| 스킬 포인트 획득 | 스킬포인트 획득                                                                           |

## 10. UI / UX 요약

### 메인 HUD

```
[상단] 코어 HP | 현재 골드 | 웨이브 정보 (N/10)
[하단] 타워 배치 버튼 | 카메라 이동 ◀ ▶ | 줌인 줌아웃
[우측] 웨이브 시작 버튼
```

### 로그라이크 선택 UI

- 웨이브 클리어 후 화면 중앙에 카드 3장 슬라이드 인
- 각 카드: 아이콘 + 이름 + 설명
- 선택 시 DOTween 연출 후 다음 웨이브 대기 화면으로

### 게임오버 화면

- 런 결과: 생존 웨이브 수, 처치 적 수, 획득 골드 합계
- 버튼: 다시 시작 / 메인 메뉴

## 12. 밸런스 조정 포인트

포트폴리오 시연 시 빠르게 재미를 보여주려면 아래 수치를 먼저 조정:

| 상황               | 조정 항목                                   |
| ------------------ | ------------------------------------------- |
| 너무 쉬움          | 적 HP 배율 상향 / 웨이브 클리어 보너스 하향 |
| 너무 어려움        | 시작 골드 상향 (150→200) / 적 스폰 수 감소  |
| 시너지가 안 느껴짐 | 시너지 배율 +0.2씩 상향                     |
| 골드가 남아돌      | 타워 업그레이드 2단계 비용 상향             |
